Decision Tree

What is a Decision Tree?

A decision tree is a decision tree that uses a series of questions and conditions to make predictions or decisions. This algorithm is used in machine learning and is built as a decision tree, each node of which has a question related to the input features, and each edge is an answer to that question. Finally, by asking questions and making decisions based on attributes, the decision tree reaches final decisions.

Support Vector Machine(SVM)

What is a Support Vector Machine(SVM)?

A Support Vector Machine (SVM) is a supervised machine learning algorithm used for classification and regression tasks. Its primary objective is to find the optimal hyperplane that best separates data points of different classes in a high-dimensional space. SVM achieves this by maximizing the margin, which is the distance between the hyperplane and the nearest data points (called support vectors) of each class. By doing so, SVM is robust to overfitting and can handle complex datasets. It works by mapping input data points into a higher-dimensional feature space using a kernel function, allowing it to find nonlinear decision boundaries. SVM is widely used in various applications such as text categorization, image classification, and bioinformatics due to its effectiveness and versatility.

K Nearest Neighbours(KNN)

What is a K Nearest Neighbours(KNN)?

The K-Nearest Neighbors (KNN) algorithm is a machine learning algorithm for classification and regression problems. In this algorithm, to predict the label or value of a new point, first the K closest points to that point in the feature space (input space) are selected. Then based on the majority of neighboring points' labels or values, the new point's label or value is estimated. This algorithm uses a simple idea to estimate the label or value of new points and is used in many application areas including pattern recognition, item sorting, and predictions.

A centroid-based algorithm(K-MEANS)

What is a centroid-based algorithm(K-MEANS)?

K-Means is a centroid-based algorithm used for clustering data into 'k' distinct groups. It begins by randomly placing 'k' centroids in the feature space. Then, each data point is assigned to the nearest centroid, forming clusters. The centroids are recalculated as the mean of the data points in each cluster, and the process iterates until convergence. K-Means aims to minimize the within-cluster sum of squared distances. While efficient and scalable, it may converge to local optima and struggle with irregularly shaped clusters.
